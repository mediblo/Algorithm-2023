# 1장 코드 내역 ( 해결한 코드만 게시 )
##### 1.1 최대 숫자 찾기개의 정렬되지 않은 숫자 중 가장 큰 값을 찾기
입력 : 정렬 되지 않은 n개의 정수 또는 실수의 배열  
출력 : 정수 또는 실수의 가장 큰 값  
사용 알고리즘 : 순차탐색  
빅오 표기법 : O(n)

##### 1.2 임의 숫자 찾기
n개의 정렬되지 않은, 또는 정렬된 숫자 중 임의 지정 숫자 찾기  
입력 : 정렬 되지 않은 또는 정렬된 n개의 정수 또는 실수의 배열, 임의 지정 숫자 K  
출력 : K 또는 찾지 못함  
사용 알고리즘 : 순차탐색 또는 이진탐색  
빅오 표기법 : O(n)

##### 1.3 거스름돈 계산
계산 후 거스름돈을 동전으로 환산 할 때 최소 갯수로 계산  
입력 : 거스름돈을 할 값  
출력 : 각 동전의 갯수 ( 500, 100, 50, 10 )  
사용 알고리즘 : 그리디  
빅오 표기법 : O(1)

##### 1.5 한붓그리기
n개의 점을 모두 지나는 방법 찾기  
입력 : n개의 점과 사이의 길  
출력 : 움직이는 경로  
사용 알고리즘 : 깊이 우선 탐색 (DFS), 재귀함수  
빅오 표기법 : O(2n)

##### 1.6 가짜 동전 찾기
n개의 무더기로 있는 동전 중 조금 더 가벼운 가짜 동전이 하나 섞여있음, 양팔 저울로 사용하여 가짜 동전 찾기  
입력 : n개의 동전  
출력 : 가짜 동전의 번호  
사용 알고리즘 : 분할 정복 알고리즘  
빅오 표기법 : O(log n)

##### 1.7 독이 든 와인 찾기
n개의 무더기로 있는 와인 중 독이 든 와인이 하나 섞여있음, 신하를 사용하여 독이 든 와인 찾기  
입력 : n개의 와인, 독이든 와인  
출력 : 죽어버린 신하의 수  
사용 알고리즘 : 2진수  
빅오 표기법 : O(log n)
