# 알고리즘 설계 구현 결과 : <달에서의 몸무게>

## 1. <문제 정의>
- **문제이름**: 달에서의 몸무게
- **문제설명**: 달에서의 몸무게는 몇일까?

## 2. <알고리즘>
- **명칭**: 달에서의 몸무게 ( moon )
- **입력**: 지구에서의 몸무게 ( kg )
- **출력**: 달에서의 몸무게 ( kg )
- **처리순서** :
	1. 지구에서의 몸무게를 입력받는다
	2. 달은 지구 표면 중력의 1/6이므로 0.17을 곱한다.

## 3. <코드 설계>
1. **자료 구조 정의**
	- 지구에서의 몸무게 : (int kg) - 정수

2. **함수 정의**
- **기능** : 지구에서의 몸무게를 통해 달에서의 몸무계를 계산하여 알 수 있다.


## <C코드 구현 결과>
헤더파일은 생략 ( stdio.h ) 
```c
int main() {
	int kg;

	printf("몸무게를 입력하세요(단위 : kg) : ");
	scanf("%d", &kg);

	printf("달에서의 몸무게는 %.2fkg입니다.", (float)kg * 0.17);
	return 0;
}
```
